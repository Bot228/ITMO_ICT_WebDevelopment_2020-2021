# List Review

Удаляет информацию об определенном авторе

**URL** : `reviews/`

**Methods** : `GET`

**Data constraints** : `{}`

**Generics type** : `ListAPIView`

## Success Responses

**Code** : `200 OK`

**Content** : `{[]}`
